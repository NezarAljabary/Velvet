# Replace App Icons

This folder is where branded app icons should live.

## Replace These Files

- `icon.ico`: Windows app and installer icon used by Electron Forge
- `icon.png`: Optional additional icon source for other tooling or documentation

## Current State

The project includes `icon.png` as a placeholder used for packaging. For production branding, replace it with a real multi-size Windows `.ico` file named `icon.ico`, and optionally keep `icon.png` as a source asset.
